WordPress 3.5 beta 3发布

发表于 2012-11-15 | 分类于 2014

来自WordPress官方的消息，WordPress 3.5 beta 3发布并提供下载试用。
和之前的两个beta版本一样，WordPress 3.5依然在测试中。
WordPress官方在3.5版本主要的工作就是重构了多媒体功能；把上传功能集成到了一个按钮，并且针对图片的排列也做了重构，增强了图片的排列功能，使图片更易于查看；

在beta1和beta2的测试中，WordPres官方也测试了大量的插件，确保之前很多插件能继续在WordPress 3.5本版上继续使用；WordPress程序和主题，插件的兼容性，始终是大问题；

从beta2开始，beta版本修正了超过300个bug；到了第三个beta版本，WordPress也应该到RC版本了；

WordPress也做了如下的更新：

jQuery UI 1.9.1；SimplePie 1.3.1, 和 TinyMCE 3.5.7。

WordPress 3.5 beta3下载地址：http://wordpress.org/wordpress-3.5-beta3.zip
